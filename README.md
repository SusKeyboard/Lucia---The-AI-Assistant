# Lucia---The-AI-Assistant
```
 Lucia is a modular personal AI assistant designed to communicate naturally with users, remember context, interact with external services, and control connected devices through an extensible plugin system.
```
A modular AI assistant built in Python.

The goal of this project is to learn modern software engineering by building an AI assistant from scratch.

Planned features:
- AI Chat
- Long-term memory
- Voice input/output
- Plugin system
- Raspberry Pi integration
- ESP32 control
- Discord support
- Telegram support
- Web interface
